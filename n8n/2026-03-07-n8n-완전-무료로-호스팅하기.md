### Context
GCP의 Always Free 티어를 활용하여 n8n을 완전히 무료로 호스팅하는 방법을 배우게 되었습니다. 이 과정에서는 Docker와 Ngrok을 사용하여 GCP의 제한 사항을 극복하고, 언제 어디서나 n8n을 실행할 수 있습니다.

### Core
1. **GCP 설정**: Google Cloud Platform에 가입하고, Always Free Tier로 컴퓨팅 인스턴스를 생성합니다.
   * Ubuntu 20.04 LTS 선택
   * f1-micro 인스턴스 유형 사용

2. **Docker 설치 및 n8n 컨테이너 배포**:
   ```bash
   sudo apt update
   sudo apt install docker.io
   sudo docker run -it --rm --name n8n -p 5678:5678 n8nio/n8n
   ```

3. **Ngrok 설정**: 외부에서 접근 가능하도록 Ngrok을 설치하고 실행합니다.
   ```bash
   wget https://bin.equinox.io/c/4VmDzA7iaHb/ngrok-stable-linux-amd64.zip
   unzip ngrok-stable-linux-amd64.zip
   ./ngrok http 5678
   ```

### Insight
이 방법을 사용하면 비용 없이 n8n의 강력한 워크플로우 자동화 기능을 활용할 수 있습니다. GCP의 Always Free Tier는 성능에는 제한이 있지만, 개인 프로젝트 또는 작은 규모의 자동화 작업에는 충분합니다. Docker를 이용해 쉽게 배포하고, Ngrok으로 간편하게 외부 접근을 설정할 수 있습니다.
